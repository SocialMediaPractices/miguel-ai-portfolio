## Asset Checklist Before Publishing

Add redacted/demo screenshots under:

```text
public/screenshots/solar-tracker/
```

Recommended screenshot set:

1. Daily summary dashboard
2. Goal progress and key metrics
3. Customer engagement entry screen
4. Customer status / engagement timeline
5. Stats graph / weekly averages
6. Objection and sentiment categories

Do not publish:

- Real customer names
- Phone numbers
- Addresses
- Private notes about people
- Account credentials
- Private repo/deployment URLs

Use demo data whenever possible.

## CLI Prototype

This repo includes a small local CLI prototype for the Solar Field Sales Intelligence Tracker case study.

```bash
npm run tracker -- seed
npm run tracker -- summary --all
npm run tracker -- export --out exports/solar-tracker-demo.csv
```

See `cli/README.md` for commands and privacy notes. Use demo data only.

## LinkedIn Drafting CLI

This repo includes a local CLI for generating LinkedIn profile and job-search copy. It does not post externally.

```bash
npm run linkedin -- headline
npm run linkedin -- all
```

Generated drafts are written to `exports/linkedin/`.
